1. `01-first-outreach.md` — a focused first-contact email.
2. `02-one-time-follow-up.md` — one respectful follow-up that makes replying easy.
3. `03-quote-confirmation.md` — a written scope-and-fee confirmation before work starts.
4. `04-decline-low-budget.md` — a clear low-budget decline with an optional smaller scope.
5. `05-retrospective-renewal.md` — an evidence-based post-campaign renewal note.

Each file contains one complete sample. Replace every `[bracketed field]`, remove anything that does not apply, and verify the recipient, claims, dates, scope, rights, fee, and payment terms before sending.
